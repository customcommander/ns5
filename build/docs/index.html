NS5 - Object Validation Made Simple

Getting Started

Loads the library: e.g.

<script src="/path/to/ns5.min.js"></script>

Quick Example

Say you want to make sure that this guy is 21 or older:

var this_guy = {
    name: 'john',
    age: 25
};

Write an adult validator: e.g.

var is_adult = new NS5({
    age: function (value) {
        return value >= 21;
    }
});

And then:

if ( is_adult.test(this_guy) ) {
    // do stuff...
}
